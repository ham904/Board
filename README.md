# 프로젝트 소개
자바 스프링 부트 환경에서 만든 게시판입니다.

스프링 mvc패턴을 이용하여 개발하였고, View 부분은 thymeleaf 템플릿 엔진을 사용하였습니다.

# 개발기간
2024.04.05 ~ 2024.05.15
